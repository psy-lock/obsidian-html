Usage

Entity Active State

On each monster's turn, all entities of that type are marked as active, visible by a red highlight. Double click on the entity to mark the end of the entity's turn.

FAQ

A player is missing for a session, how do we keep the character in the party without participating in the scenario?

There is an Absent feature. Mark a character as absent by opening the character menu or character sheet and just clicking on the character name

Shortcuts

Utilities

  • CTRL + z: Undo
  • CTRL + y | CTRL + SHIFT + z: Redo
  • ESC: open main menu
  • ↑ | +: Zoom-In
  • ↓ | -: Zoom-Out

Перебіг Гри

  • ↹: toggle active figure when drawn / toggle initiative input when focused input
  • n: next round / draw
  • m: draw AM card (deck depending on active figure, monster deck when not active, disabled when Draw)
  • a draw with Advantage
  • d draw with Disadvantage
  • 1-6: toggle through element state

Campaign

  • f: (finish) scenario menu
  • p: open party sheet
  • g: open global map
  • c: open scenario chart
